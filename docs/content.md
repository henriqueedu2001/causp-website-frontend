# Conteúdo do Site do CAUSP
Este documento organiza todas as páginas do site do CAUSP, para melhor navegação do usuário e para facilitar a manutenção do site.

## Páginas

### Home
Página inicial, de apresentação do coletivo. Ela possui as seguintes secções:

- Logotipo do CAUSP
- Quem Somos
- Nossa história
- Descubra
- Contatos

### Quem Somos
Página de descrição do coletivo, com a definição institucional, membros, resumo histórico, missão, visão, valores.

- Definição Institucional
- História (resumido)
- Missão
- Visão
- Valores
- Membros

### Nossa história
Página de história do CAUSP desde sua fundação e resumo de suas principais conquistas.

- Fundação
- Primeiros Anos
- Conquistas

### Descubra
Página com conteúdo informativo produzido pelo CAUSP, como cartilhas explicativas, pesquisas, artigos, textos etc.

- Materiais científicos sobre TEA

### Cultura
Página com conteúdos culturais produzidos pelo CAUSP, como artes, textos, etc.

- CAUSP artes
- Teatro Atípico

### Participe
Página de inscrição para participar do CAUSP.

- Regimento do CAUSP
- Formulário

## Barra de Navegação
A Barra de navegação conta com os seguintes itens:
1. **Quem Somos?**
2. **Nossa História**
3. **Descubra!**
4. **Cultura**
5. **Participe!**
6. **Siga-nos!**
